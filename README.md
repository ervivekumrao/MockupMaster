# Mockup Master: Features & User Guide

<!-- Copyright 2025 Vivek Umrao -->

Welcome to the comprehensive guide for **Mockup Master: Dev Tool**. This document covers everything from the core features to step-by-step instructions on how to use the tool effectively.

---

## 🚀 Key Features

### 📱 Contact Generation
- **Realistic Data:** Generates contacts with full names, multiple phone numbers (mobile, home, work), and email addresses.
- **Avatars:** Automatically assigns placeholder avatars to make the contact list look authentic.
- **Bulk Creation:** Create 10, 50, or 100+ contacts in a single tap.
- **Organization:** Contacts are added to a specific "Mockup Master" account for easy identification.

### 💬 SMS Mocking
- **Full Thread Simulation:** Create entire conversation histories between different numbers.
- **Message Types:** Supports Sent, Received, and Draft messages.
- **Status Simulation:** Mock "Read," "Delivered," and "Sent" statuses.
- **MMS Support:** Simulate messages with attachments and multi-recipient threads.

### 📞 Call Log Generation
- **History Variety:** Generate logs for Incoming, Outgoing, Missed, Rejected, and Blocked calls.
- **Custom Durations:** Randomize call lengths to simulate real-world usage.
- **Contact Integration:** Links call logs to existing contacts on the device.

### 📂 Data Portability
- **Export Formats:** Save your mock datasets in **JSON**, **CSV**, or **XML**.
- **Cross-Device Testing:** Export data from one emulator and import it into another.

### 🧹 One-Tap Cleanup
- **Safe Deletion:** Tracks every entry created by the tool and allows for a surgical wipe, leaving your personal data untouched.

## 📖 How to Use

### 1. Initial Setup & Permissions
Upon first launch, the app will request access to **Contacts**, **SMS**, and **Call Logs**.
- Tap **"Grant Permissions"** on the home screen.
- **IMPORTANT:** To write SMS data on Android 4.4+, you must temporarily set Mockup Master as your **Default SMS App** when prompted.

### 2. Generating Mock Data
1. **Navigate:** Use the bottom navigation or tabs to select **Contacts**, **SMS**, or **Call Logs**.
2. **Configure:** Select the quantity and any specific presets (e.g., "Corporate" or "Heavy User").
3. **Generate:** Tap the **Generate** button. A progress bar will indicate the status.

### 3. Exporting & Importing
- **To Export:** Go to the **Portability** section, select a format (JSON/CSV/XML), and save the file.
- **To Import:** Tap **"Import Data"**, select a previously exported **JSON** file, and the app will recreate the data on the current device.

### 4. Cleaning Up the Device
To remove all data generated by this tool:
1. Tap the **"Cleanup"** or **"Clear All"** button on the main dashboard.
2. Confirm the action to delete only the tracked mock entries.

---

## 💡 Developer Tips
- **Consistent Testing:** Use the Export/Import feature to share specific data scenarios across your QA team.
- **UI Stress Testing:** Use the "Heavy User" preset to see how your own app handles thousands of contacts or massive SMS threads.
- **Default SMS App:** Don't forget to switch back to your preferred SMS app after generating mock messages.
